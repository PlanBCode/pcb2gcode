This tool is an attempt to make the process of generating gcode from pcb designs more accessible. It is basically a, hopefully easy to use, interface that uses the pcb2gcode program to generate gcode.

You need gcode of pcb's when you want to do pcb milling.

This tool is a initiative of .. at Fablab Amersfoort. It is a opensource project and you can find the source on the following page.
github.com/PlanBCode/pcb2gcode
Your suggestions and issues are welcome, you can leave them on the following page.
github.com/PlanBCode/pcb2gcode/issues
